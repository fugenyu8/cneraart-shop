# 源・华渡电商系统 - 完整工作清单

## 📌 项目概述
- **项目名称**: 源・华渡 (Yuan · Huadu)
- **目标用户**: 欧洲华人和对东方文化感兴趣的西方人
- **主要语言**: 英文(默认) + 中文
- **部署平台**: Railway
- **技术栈**: React + TypeScript + Node.js + MySQL + tRPC

## 🎨 设计要求

### 视觉风格
- **东方禅意 + 现代高贵**
- 深蓝/黑色背景
- 金黄色/深红色点缀(佛教僧袍色)
- 五台山寺庙背景图片
- 佛教元素融入(观音、文殊菩萨、释迦牟尼佛等)
- 书法字体标题

### 关键元素
- 五台山高僧开光认证
- 文殊菩萨道场
- 五爷庙(财神)
- 神秘庄重的氛围
- 不要开光证书相关内容

### Logo
- 中文: "源・华渡" (马善政书法字体)
- 英文: "YUAN · HUADU" (Cinzel字体)
- 金黄色配色

### 字体大小
- 导航栏Logo: text-lg
- Hero标题: text-2xl/4xl/5xl (响应式)
- 避免过大,确保中英文切换不拥挤
